# Output Format

Return exactly this shape. Nothing before it, nothing after.

---

**Drop point:** [line/timestamp] then a one-line quote of the line.
**Why it dropped:** one sentence.

**The five reads**

| Read | Score | The gap vs your best close |
|---|---|---|
| Objection trail | x/5 | ... |
| Price moment | x/5 | ... |
| Next-step lock | x/5 | ... |
| Talk balance | x/5 | ... |
| Momentum | x/5 | ... |

**Unhandled threads:** bullet list, each with its line cite.

**The follow-up line to send:**
> the message, ready to copy.

---

Rules for the report:
- Every score row names a concrete gap against a real line in `context/best-closes.md`.
- If the call closed, replace "Why it dropped" with "What carried it" and report the win so the owner can repeat it.
- If `best-closes.md` was empty, add one line at the top: "Scored against the default standard. Fill best-closes.md to score against your own calls."
