# The Five Reads — Scoring Rubric

The Drop-Point Read scores one call across five reads. Each read runs 0 to 5 and is scored against the owner's own winning calls in `context/best-closes.md`, not a generic playbook. If best-closes is empty, score against the default anchors below and say so.

The headline output is the drop point: the single line where momentum fell. The five reads are the supporting read.

## How to score
For each read: pick the score, then name the one line from `best-closes.md` where the owner did this better. The gap has to be concrete, tied to a real line in their own winning call.

---

## 1. Objection trail
**Measures:** every hesitation the buyer voiced, and whether each got answered and then reopened to confirm it landed.
- **0-1:** A concern got one fast answer and never came back.
- **2-3:** Concerns answered in the moment, none circled back to.
- **4-5:** Each concern answered, then circled back to confirm it landed.

## 2. Price moment
**Measures:** how price entered the call and whether value got requalified around it.
- **0-1:** Number dropped, buyer went quiet, call moved on.
- **2-3:** Price stated with a weak tie to value.
- **4-5:** Price framed against the buyer's own stated cost of staying stuck.

## 3. Next-step lock
**Measures:** a concrete, scheduled next action before hangup.
- **0-1:** "I'll follow up" with no date.
- **2-3:** A next step named, no date set live.
- **4-5:** Calendar hold set live on the call.

## 4. Talk balance
**Measures:** buyer share of the conversation during discovery.
- **0-1:** Rep talked 70% or more.
- **2-3:** Roughly even, rep led most threads.
- **4-5:** Buyer talked through their own problem.

## 5. Momentum
**Measures:** the point engagement dropped and whether it recovered.
- **0-1:** One clear drop, no recovery.
- **2-3:** A dip that partly recovered.
- **4-5:** Energy held or climbed to close.

---

## The drop-point rule
The drop point is almost never where the call ended. It is the quiet line minutes earlier that made the ending inevitable. Name one. Give the line or timestamp. Quote it. Everything after it was a formality, so find the moment, not the symptom.
